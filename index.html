Projects

Text Sentiment Analysis

This project applies machine learning–based sentiment analysis to Twitter data by transforming raw tweets into numerical features and training several classifiers to predict sentiment labels. Text is first cleaned and converted into vectors using Bag‑of‑Words and TF‑IDF. Four algorithms are trained: Linear SVC, Complement Naive Bayes, Multinomial Naive Bayes, and Logistic Regression. The experiments show performance depends on both the vectorizer and the classifier.
