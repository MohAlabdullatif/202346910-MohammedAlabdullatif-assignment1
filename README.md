# Personal Portfolio Website

A responsive personal portfolio website built using HTML, CSS, and JavaScript.  
The website presents my background, projects, achievements, and contact information in a clean and modern layout.

---

## Features

### Responsive Design
- Fully responsive layout using Flexbox
- Optimized for desktop, tablet, and mobile devices
- Uses `clamp()` for scalable typography

### Dark / Light Theme Toggle
- Toggle button in the header
- Switches between light and dark mode dynamically
- Implemented using JavaScript class toggling

### Dynamic Greeting Message
- Displays greeting based on time of day:
  - Good Morning 
  - Good Afternoon 
  - Good Evening
- Automatically generated using JavaScript Date object

### Structured Sections
- About Me
- Projects
- Achievements
- Contact Information

### Smooth Navigation
- Sticky header
- Smooth scrolling behavior
- Internal anchor navigation
